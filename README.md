# SunnyLand
**SunnyLand** is a cross-platfrom platformer game developed in C++ with SDL3, glm, nlohmann-json and Tiled.

## Control
```
A,D / left,right - to move;
W,S / up,down - to climb;
J / Space - to jump;
P / ESC - show menu and pause;
F4 - toggle fullscreen;
```
